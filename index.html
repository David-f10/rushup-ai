04 Mode d'analyse Choisis un mode
Mode Questions
Je connais mon angle
Tu décris l'angle éditorial et les thèmes. L'IA structure selon le framework intro/tension/développement/conclusion.
Mode Découverte
Je découvre le sujet
L'IA analyse et détecte les thématiques. Tu choisis, elle sélectionne et adapte par canal.
Mode Script
Nettoyer les prises
L'IA retrouve les meilleures prises dans l'ordre original. Idéal pour les face cam, prompteur, texte répété en plusieurs prises.
L'IA analyse le rush, identifie jusqu'à 8 thématiques génériques, et te les présente avec un score d'intensité. Tu choisis, elle sélectionne et adapte pour chaque canal.
L'IA lit l'ensemble du rush, détecte les répétitions et hésitations, et reconstruit le discours le plus propre dans l'ordre original. Elle ne réordonne rien — elle sélectionne la meilleure prise de chaque passage.
Sans script : l'IA détecte automatiquement le message et nettoie les prises. Avec script : elle aligne chaque ligne du script avec la meilleure prise du rush.
